## Overview

A complete e-commerce application built with Java, Spring Boot, and Angular, featuring product management, shopping cart, secure checkout, and an administrative dashboard.

## Main Features

- **Product catalog**: Category-based browsing, search, and filtering with paginated results
- **Shopping cart**: Session-based cart with quantity management
- **Checkout flow**: Multi-step checkout with address, shipping, and payment integration
- **Admin dashboard**: Product CRUD, order management, and customer analytics

## My Contribution

Developed the full-stack application: Spring Boot REST APIs for product, cart, and order management, Angular frontend with responsive UI, and MySQL database design with optimized queries.

## Challenges

Implementing a secure checkout flow with payment gateway integration, managing cart state across sessions, and designing an efficient product search with filtering across categories.
