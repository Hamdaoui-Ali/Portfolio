## Overview

A modular web scraping and automation toolkit that extracts structured data from websites, processes it through transformation pipelines, and delivers results via REST APIs.

## Main Features

- **Multi-source scraping**: Configurable scrapers for diverse website structures
- **Data transformation**: Pipeline for cleaning, normalizing, and enriching scraped data
- **REST API delivery**: FastAPI endpoints for querying and retrieving processed data
- **Scheduled execution**: Automated scraping jobs with configurable intervals

## My Contribution

Designed the modular scraper architecture with base classes for extensibility. Built the FastAPI REST layer, implemented the data transformation pipeline, and wrote comprehensive tests with PyTest.

## Challenges

Handling dynamic content and anti-bot mechanisms, maintaining scraper reliability across website changes, and ensuring data quality through the transformation pipeline.
