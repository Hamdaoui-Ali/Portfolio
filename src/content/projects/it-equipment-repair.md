## Overview

An enterprise-grade system for managing IT equipment repair workflows, tracking service history, and monitoring inventory across departments.

## Main Features

- **Repair ticketing**: Create, assign, and track repair requests with status workflow
- **Service history**: Complete audit trail of repairs per equipment item
- **Inventory management**: Real-time tracking of equipment stock and spare parts
- **Reporting dashboards**: Repair turnaround metrics, technician performance, and cost analysis

## My Contribution

Architected the full application with Java and Spring Boot. Implemented the repair ticketing workflow engine, inventory management module, and reporting dashboards. Containerized with Docker for consistent deployment.

## Challenges

Designing a flexible repair status workflow that supports different equipment types, implementing efficient inventory tracking with concurrent updates, and building intuitive dashboards for IT support teams.
