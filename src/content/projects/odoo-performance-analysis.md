## Overview

A prototype tool for analyzing and benchmarking Odoo ERP module performance, helping teams identify bottlenecks and optimize large-scale deployments.

## Main Features

- **Module benchmarking**: Measures load times and resource usage per Odoo module
- **Bottleneck detection**: Identifies slow queries, heavy modules, and performance regressions
- **Optimization recommendations**: Generates actionable suggestions based on analysis results
- **API testing**: Postman collections for automated ERP endpoint validation

## My Contribution

Built the analysis engine with Python and Flask, developed benchmarking scripts for Odoo modules, and created automated API test suites with Postman and PyTest.

## Challenges

Accurately measuring Odoo module performance without affecting production systems, and building a flexible analysis framework that adapts to custom module configurations.
