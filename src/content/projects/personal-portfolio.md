## Overview

A modern, performant personal portfolio built as a static site with Astro, showcasing projects, skills, and professional experience.

## Problem

Full-Stack Software Engineers need an online presence that demonstrates technical credibility, showcases real project work, and communicates professional identity clearly to recruiters and collaborators.

## Solution

Built with Astro for static site generation, Tailwind CSS for styling, and TypeScript for type safety. Features content collections for project management, dynamic routing for project detail pages, and structured SEO metadata. Deployed automatically via Vercel with GitHub Actions CI/CD.

## Main Features

- Static site with 12 pages (homepage + 11 projects)
- Content Collections for schema-validated project data
- SEO metadata, JSON-LD structured data, and sitemap
- Dark/light/system theme switcher with ARIA accessibility
- Migration pipeline and before/after comparison components
- Technology stack showcase with categorized groups

## My Contribution

Sole developer for the entire project. Designed the information architecture, implemented all components, set up CI/CD, and authored all content.

## Challenges

Balancing visual polish with static site performance, designing reusable components for project pages, and ensuring accessibility compliance across all interactive elements.
